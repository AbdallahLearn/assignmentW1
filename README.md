# Advanced User Registration System

## Project Overview
This project is an advanced user registration system inspired by Salamah's Company Registration System. The system provides a fully responsive, animated, and validated registration portal that integrates external API calls and utilizes modern web development technologies.

## Technologies Used
- **Frontend:** HTML5, TailwindCSS, Material UI, Animate.css
- **JavaScript:** ES6+, Promises, Async/Await, Modules, Component-Based Architecture
- **TypeScript**
- **Version Control:** GitHub
- **API Integration:** Fetch API for retrieving company registration details

## Features Implemented
### 1. HTML5 & Basic Structure (10 Marks)
- Structured landing page with semantic elements.
- Registration form including:
  - Company Name
  - Commercial Registration Number
  - Email & Phone Number
  - Password & Confirm Password
  - Address Fields (City, Region, Zip Code)
  - Business Type (Dropdown Selection)
  - Terms & Conditions Checkbox
  - Submit Button

### 2. CSS Styling (15 Marks)
- **TailwindCSS**: Utility-based styling for fast and consistent design.
- **Material UI**: Styled components for form inputs and buttons.
- **Animate.css**: Smooth animations for form elements (fade-in, input focus effects).
- **Responsive Layout**: Implemented with Flexbox & CSS Grid.

### 3. JavaScript Validation (20 Marks)
- Client-side form validation to ensure all fields are completed.
- Email validation and password strength verification.
- Error messages displayed for incorrect inputs.
- Form submission handling using JavaScript Fetch API.
- Utilized Promises & Async/Await for API calls.
- ES6+ features like Arrow functions and Modules.

### 4. API Integration (25 Marks)
- Integrated an external API to fetch company registration details.
- Used **fetch()** with Async/Await to retrieve and display company data dynamically.
- Implemented proper error handling for API responses.

## Bonus Features (Extra 10 Marks)
- OTP verification before form submission.
- Dark mode toggle implemented using JavaScript.
- Custom loading spinner displayed during API calls.

## Setup Instructions
1. Clone the repository from GitHub:
   ```sh
   git clone https://github.com/your-username/registration-system.git
   ```
2. Navigate to the project directory:
   ```sh
   cd registration-system
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
5. Open `index.html` in a browser to view the registration system.
